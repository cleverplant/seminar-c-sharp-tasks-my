
static - это чтобы void существовал в единственном экземпляре
и не пришлось бы объявлять экземпляр класса

# region
здесь можно группировать методы
# endregion

main - если мы работаем внутри функции main тогда нельзя описывать методы с одним именем, 
но если мы выйдем за метод main мы можем описывать методы с одинаковыми именами

в одном пространстве но с одним именем могут быть - если не внутри одной функции

метод main - вообще нужен для запуска программы - это точка входа -она запускает программу

если удалить метод main - это просто бибка(сборище файлов) 
если вернуть - получится программа

!!! в библиотеке нет способа запстится - все бибки это прицеп к основной программе, которую и описывает main !!!
